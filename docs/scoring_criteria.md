# 超级AI大学人格测试 · 评分与划分标准

## 一、测试架构概览

本测试共 **4 个核心维度**，总计 **31 道情境题**，每题 4 个选项，覆盖港科广学生的真实校园记忆与科研日常。

| 维度 | 名称 | 两极 | 题目数 |
|------|------|------|--------|
| 维度一 | 充电模式 | E（社交电源） vs I（独行电池） | 7 |
| 维度二 | 脑回路模式 | S（细节侦探） vs N（脑洞建筑师） | 7 |
| 维度三 | 执行节奏 | J（计划仙人） vs P（DDL战神） | 7 |
| 维度四 | AI协作模式 | A（AI原生派） vs H（古法科研派） | 10 |

> 最终人格由 **4 个维度的倾向组合** 而成，共 `2^4 = 16` 种人格类型。

## 二、计分规则

### 2.1 单题计分

每道题的 4 个选项按「光谱」排列，对应不同维度的倾向强度：

| 选项 | 倾向强度 | 计分方式 |
|------|----------|----------|
| A | 极端左极 | 左极 +3，右极 +0 |
| B | 温和左极 | 左极 +1，右极 +0 |
| C | 温和右极 | 左极 +0，右极 +1 |
| D | 极端右极 | 左极 +0，右极 +3 |

**示例**：维度一（E-I）某题
- 选 A → `{ E: 3 }`
- 选 B → `{ E: 1 }`
- 选 C → `{ I: 1 }`
- 选 D → `{ I: 3 }`

### 2.2 维度总分计算

将某维度下所有题目得分分别累加至两个方向，例如：

```
维度一（E-I）总分：
  E = 3 + 1 + 3 + 0 + 1 + 3 + 0 = 11
  I = 0 + 0 + 0 + 3 + 1 + 0 + 3 =  7
```

### 2.3 倾向判定

比较该维度两个方向的得分：
- **得分高者** 即为该维度的倾向
- 若两者 **得分相同**，默认取左极（E / S / J / A）

> 后续可扩展「平衡态」标签，当两方向分差 ≤ 2 时，结果描述中标注「XX倾向不明显，处于平衡态」。

### 2.4 人格代码组合

将 4 个维度的倾向按顺序拼接，得到 4 位人格代码：

```
人格代码 = [E/I] + [S/N] + [J/P] + [A/H]
```

**示例**：
- E 得分 > I 得分 → 取 `E`
- S 得分 < N 得分 → 取 `N`
- J 得分 > P 得分 → 取 `J`
- A 得分 > H 得分 → 取 `A`
- **最终人格**：`ENJA`

## 四、人格描述设计规范

每种人格描述遵循以下结构：

1. **开篇定调**：用一句话概括「你就是这样的人」，不评判对错
2. **校园梗埋设**：融入只有本校生懂的场景（如红鸟广场、秋秋、图书馆三楼、早九工商银行等）
3. **四维度解析**：简要说明该人格在四个维度上的典型表现
4. **日常技能 / 隐藏 Buff**：赋予一个游戏化的能力标签，增加传播性
5. **同频建议**：给出一句温暖的人际建议

### 描述示例（ENJA · AI乌托邦架构师）

> **开篇**：你是那个在社团群里一边用AI生成活动流程图、一边把音乐节摊位布置得井井有条的人。你不仅享受和人碰撞想法，更相信技术能让这些想法飞得更高。
>
> **校园日常**：你很可能在图书馆三楼靠窗的位置，一边用GPT整理文献综述，一边在笔记本上画着下学期的创业蓝图。红鸟广场的夕阳对你来说不是风景，而是灵感触发器。
>
> **隐藏 Buff**：「蓝图预言」——当你用AI辅助制定计划时，方案成功率 +30%。
>
> **同频建议**：偶尔也让计划表空白几个小时，给意外的美好留一条缝隙。

## 六、扩展建议

1. **最小可行版本**：若需快速上线，可从每个维度抽取 2 题（共 8 题），人格体系不变，后续迭代补充题目。
2. **动态题目池**：未来可支持从题库中随机抽题，每次测试题目不同但维度覆盖完整。
3. **结果页维度条**：结果页建议展示 4 个维度的「倾向对比条」（如 E ████████░░ I），让用户直观看到自己的光谱位置，而非仅给一个标签。
4. **稀有度机制**：可统计各人格类型的测试占比，给较少见的人格标注「稀有」「传说」等标签，增加分享动力。
