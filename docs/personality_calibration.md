# 人格结果校准表（第一步）

> 目的：以当前稳定的四字母判定逻辑为核心，审视 16 型人格的“强记忆点人设”是否真正从字母定义自然生长出来，而不是仅仅“好玩但未必贴码”。

---

## 一、校准基准

当前项目的底层逻辑以四个维度为准：

- `E / I`：社交电源 / 独行电池
- `S / N`：细节侦探 / 脑洞建筑师
- `J / P`：计划仙人 / DDL 战神
- `A / H`：AI 原生派 / 古法科研派

本轮校准默认以下原则成立：

1. **题目与计分逻辑暂不调整**
2. **人格代码仍由四个维度直接拼接得到**
3. **结果文案必须能够被“反向读回”四个字母**

也就是说，用户看完某型人设后，应该能大致感知：

- 这是更偏外向还是内向
- 这是更偏细节还是脑洞
- 这是更偏规划还是临场
- 这是更偏 AI 协作还是更偏人脑主导

---

## 二、总体判断

### 1. 当前系统中，**题目与字母逻辑是稳定的**

问题不在测试怎么测，而在结果文案怎么解释。

### 2. 当前人格文案的主要偏差，不是“算错”，而是“写偏”

具体表现为：

- 有些人设先有强记忆点，再硬挂到四字母组合上
- 有些文案只突出一个梗，没有把四个字母都显形
- `A / H` 维度整体最容易被遮住，是当前最需要补强的一维

### 3. 这轮校准的目标

不是把文案写得更“标准 MBTI”，而是让它同时满足：

- **能传播**
- **有校园记忆点**
- **能解释为什么是这四个字母**

## 四、逐型校准表

### ESJA

- 四字母骨架：外向、细节导向、强规划、把 AI 当自然协作搭子
- 当前人设：`全能ACE / 校园六边形战士`
- 贴合度：高
- 主要优点：
  - `E` 很明显，社交、组局、场域掌控力强
  - `S/J` 很明显，提前交作业、准备充分、执行稳定
- 主要问题：
  - `A` 只是隐含存在，没有形成鲜明的“AI 原生协作”特征
- 建议方向：
  - 保留“六边形战士”
  - 在文案里补上“会自然用 AI 整理、统筹、提效”的行为细节

### ESJH

- 四字母骨架：外向、细节导向、强规划、偏传统/人脑主导
- 当前人设：`摆渡人 / 总把别人送到对岸的人`
- 贴合度：中
- 主要优点：
  - `E` 很清晰，愿意与人高频互动
  - 有服务型、稳定型人格魅力
- 主要问题：
  - 当前文案更像“高共情照顾者”，而不是“细致、规划、古法”的执行型人格
  - `H` 几乎没有落地到学习/做事方式
  - `S/J` 的秩序感、流程感不够
- 建议方向：
  - 保留“照顾人、可靠”的气质
  - 往“靠谱组织者 / 稳定统筹者 / 手把手带人前进的人”方向收束
  - 明写其偏好亲自确认、自己理清、传统推进

### ESPA

- 四字母骨架：外向、细节导向、随性临场、AI 协作积极
- 当前人设：`红鸟街溜子 / 永远在串门的校园情报站`
- 贴合度：中上
- 主要优点：
  - `E` 非常强，社交和流动感都很到位
  - `P` 也明显，整个人设有即时性和现场感
  - `S` 作为“活地图”“消息集散中心”是成立的
- 主要问题：
  - `A` 没有真正显形
  - 文案更像“社交牛人”，还不够像“善于即时借助 AI 的高频连接者”
- 建议方向：
  - 保留“校园情报站”这个强记忆点
  - 增加其会顺手用 AI 搜信息、理活动、出文案、搞整活的细节

### ESPH

- 四字母骨架：外向、细节导向、随性临场、偏传统/现实主导
- 当前人设：`进击的饿殍 / 先吃饱，再谈理想`
- 贴合度：中下
- 主要优点：
  - `S` 有一定成立空间，关注口味、窗口、时间段等具体体验
  - `P` 的即时性、跟着当下状态走，也能读出来
- 主要问题：
  - 人设几乎完全被“吃货梗”占据
  - `E` 不够强，`H` 几乎不可见
  - 看完更容易记住“爱吃”，不容易读出“社交型、现实型、临场型、偏传统”
- 建议方向：
  - 可以保留“食堂/饭搭子”方向，但不能只写食欲
  - 更适合改成“现实派社交行动者 / 接地气的饭局发动机”

### ENJA

- 四字母骨架：外向、脑洞导向、强规划、AI 协作积极
- 当前人设：`预言家 / 总比别人早看见半步未来`
- 贴合度：中上
- 主要优点：
  - `N/J` 很强，未来感和路径感都比较准
  - `E` 也能从“主动入局、改写剧本”里读出来
- 主要问题：
  - `A` 没有充分显形
  - 现在更像“强势的直觉派领导者”，还不够像“AI 加持下的未来架构者”
- 建议方向：
  - 保留“预言家”的记忆点
  - 把“借助 AI 迅速搭骨架、推演、试错”写进去

### ENJH

- 四字母骨架：外向、脑洞导向、强规划、偏传统/人脑主导
- 当前人设：`弄潮者 / 总站在风口前半步`
- 贴合度：低
- 主要优点：
  - `E/N` 基本成立，确实有前瞻和号召感
- 主要问题：
  - “新的 App、新的热点、新的机会”这一套语言明显更偏 `A`
  - 当前人设是“追新者/趋势玩家”，不是“古法主导的前瞻型规划者”
  - `H` 与当前叙事冲突最明显
- 建议方向：
  - 不建议继续沿“新 App / 风口玩家”写
  - 更适合改成“靠自己调研、判断、打磨方案的未来策士 / 路演型思想领队”

### ENPA

- 四字母骨架：外向、脑洞导向、随性临场、AI 协作积极
- 当前人设：`无相之神 / 不被定义的自由流体`
- 贴合度：低
- 主要优点：
  - 有自由、松弛、不受规训的气质
- 主要问题：
  - 文案主动把 `E/I` 与 `J/P` 写成“都行”，直接消解了人格代码
  - 现在不是“这四个字母的表现”，而是“拒绝被四个字母定义”
  - `A` 也没有显形
- 建议方向：
  - 建议整体重写
  - 可保留“自由流体”的手感，但必须明确这是一个**外放、脑洞、临场、AI 原生**的人
  - 更适合往“灵感串场王 / 即兴脑洞发动机”方向收束

### ENPH

- 四字母骨架：外向、脑洞导向、随性临场、偏传统/人脑主导
- 当前人设：`假面影帝 / 线上线下像两个人`
- 贴合度：中
- 主要优点：
  - `E/N/P` 能读出来，尤其是“吸收碎片信息”“深夜熔炼成想法”这部分
  - 有鲜明的戏剧感和记忆点
- 主要问题：
  - `H` 仍不清楚
  - “线上线下像两个人”更像表现方式，不是人格结构本身
- 建议方向：
  - 保留“白天社交吸能、夜里脑洞爆发”的反差
  - 补上其更依赖自己消化、自己酝酿，而非 AI 预处理的 `H` 特征

### ISJA

- 四字母骨架：内向、细节导向、强规划、AI 协作积极
- 当前人设：`人机 / 情绪极稳的省电模式`
- 贴合度：中上
- 主要优点：
  - `I/J` 非常明显
  - `S` 也能从规则感、精密感中读出
  - 人设辨识度强
- 主要问题：
  - `A` 没有真正写出来
  - “人机”更像气质标签，不是完整的行为画像
- 建议方向：
  - 保留“省电模式”“精密运转”
  - 增加其会把 AI 作为高效工具链一部分的细节

### ISJH

- 四字母骨架：内向、细节导向、强规划、偏传统/人脑主导
- 当前人设：`老人 / 二十岁的身体，六十岁的作息`
- 贴合度：高
- 主要优点：
  - `I/S/J/H` 基本都能顺畅读出来
  - 规律、稳定、低波动、慢节奏，非常自洽
- 主要问题：
  - `H` 仍然可以再向“相信自己慢慢磨出来”补一笔
  - 目前更多是生活作息像老人，不完全是学习/做事方式像 `H`
- 建议方向：
  - 基本不用大改
  - 只需轻微补强其传统、手工、亲自验证的做事方式

### ISPA

- 四字母骨架：内向、细节导向、随性临场、AI 协作积极
- 当前人设：`DDL战神 / 死线前爆发的极限艺术家`
- 贴合度：中上
- 主要优点：
  - `I/P` 极强，是目前辨识度最高的人设之一
  - `S` 也成立，因为其临门一脚的执行仍然偏具体任务和现实交付
- 主要问题：
  - `A` 还没有被写出来
  - 当前更像“拖延型生产者”，而不是“会借助 AI 把死线操作做到极致的人”
- 建议方向：
  - 保留“DDL战神”
  - 增加其用 AI 辅助补全、提速、兜底的操作细节

### ISPH

- 当前人设：`观鸟达人 / 会为一只鸟停下一整个下午`
- 贴合度：中上
- 主要优点：
  - `I/S/H` 很强，安静观察、具体感知、长期专注都很准
  - 人设新鲜、有校园感，也不俗套
- 主要问题：
  - `P` 比较弱，当前更像“宁静而稳定”，不够像“随性、让节奏流动”
- 建议方向：
  - 保留“观鸟达人”

  - `A` 几乎没有显性信号
- 建议方向：
  - 保留“女巫”的独特质感
  - 但要把她写成“会用 AI 做推演/试药/搭原型的系统型脑洞者”，而不是纯手工炼金

### INJH

- 主要优点：

  - 观察、分析、沉浸式思考感都很强
- 主要问题：
  - `J` 还不够强，当前更像“观察者”而不是“有结构地观察和归纳”

### INPA

### INPH

## 六、下一步建议

第二步可以直接进入“人格文案重构草案”，按以下顺序推进：

1. 先改 `ENJH`、`ENPA`
2. 再改 `ESPH`、`INPH`
3. 再统一补强全体人格的 `A / H` 显性信号
4. 最后同步 `data/personalities.json` 与 `docs/scoring_criteria.md`

如果继续推进，建议下一份产出改为：

> **16 型人格文案重构方案（第二步）**

其中每型给出：

- 推荐新方向
- 推荐名称 / 标题（若需）
- 一句话骨架
- 记忆点人设
- 必须保留的四字母信号
